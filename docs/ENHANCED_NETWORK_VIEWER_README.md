# Покращений інтерактивний візуалізатор мережі

## Швидкий старт

Покращений інтерактивний візуалізатор мережі з динамічною візуалізацією та автоматичною синхронізацією з БД.

## Нові можливості

✅ **Форми вузлів залежно від кількості зв'язків**
- Коло (d≤2), Трикутник (d=3), Квадрат (d=4), П'ятикутник (d=5), Шестикутник (d≥6)

✅ **Покращена інтерактивність**
- Підсвічування при наведенні
- Перетягування вузлів
- Виділення пов'язаних вузлів
- Масштабування колесиком миші

✅ **Автоматична синхронізація з БД**
- Збереження в .db файли
- Функція `render_network_from_db()`
- Словникове сховище Python

✅ **Плавні анімації**
- Анімоване оновлення
- Ефекти виділення
- Динамічне оновлення форм

## Використання

### Запуск основного додатку
```bash
python main.py
```

### Перехід до візуалізації
1. Запустіть додаток
2. Перейдіть на вкладку "Панель візуалізації" → "Топологія"
3. Використовуйте нові можливості візуалізації

### Тестування
```bash
python scripts/tests/test_enhanced_network_viewer.py
```

### Демонстрація
```bash
python scripts/demos/demo_enhanced_network_viewer.py
```

## Режими редагування

- **Перегляд** - виділення, перетягування, масштабування
- **Додати вузол** - клік по холсту
- **Додати зв'язок** - клік по двох вузлах
- **Видалити** - клік по елементу

## Функції

- Аналіз зв'язності мережі
- Експорт для аналізу надійності
- Автоматичне збереження змін
- Підтримка всіх існуючих функцій

## Переваги

1. **Візуальна інформативність** - форми показують важливість вузлів
2. **Зручність** - покращена інтерактивність
3. **Надійність** - автоматичне збереження
4. **Красивий інтерфейс** - плавні анімації
5. **Масштабування** - детальний аналіз
6. **Аналіз хрупкості** - швидке визначення критичних вузлів

## Технічні деталі

- **Мова**: Python 3.x
- **GUI**: tkinter + matplotlib
- **База даних**: shelve (словникове сховище)
- **Графіки**: networkx
- **Анімації**: matplotlib

## Сумісність

Повністю сумісний з базовим інтерактивним візуалізатором:
- Всі існуючі функції збережені
- Зворотна сумісність
- Fallback на базовий візуалізатор при помилках
